# ⚡ ProdDash

A modern productivity dashboard built with **HTML, Tailwind CSS, and Vanilla JavaScript**. It brings together essential productivity tools in one place with a clean interface, weather integration, automatic data persistence, and a responsive layout.


---

## ✨ Features

### 🏠 Dashboard

- Personalized greeting based on current time
- Live date and digital clock
- Current weather information
- Automatic location detection
- Dynamic weather icons
- Dynamic background based on time and weather
- Responsive dashboard layout

---

### 🌦 Weather

- Current temperature
- Feels like temperature
- Humidity
- Wind speed
- Pressure
- Visibility
- Weather condition
- Automatic city detection using Geolocation API

---

### ✅ Todo List

- Add tasks
- Edit tasks
- Delete tasks
- Mark completed
- Local Storage support
- Responsive interface

---

### 📅 Daily Planner

- 24-hour planning schedule
- Auto-save while typing
- Local Storage support
- Scrollable planner section
- Responsive layout

---

### 🎯 Daily Goals

- Add goals
- Mark goals as completed
- Delete goals
- Goal statistics
- Local Storage support
- Responsive design

---

### 🍅 Pomodoro Timer

- Start timer
- Pause timer
- Reset timer
- Simple distraction-free interface

---

### 🕒 Date & Time

- Live digital clock
- Current date
- Day of the week

---

### 🌙 Theme

- Light Mode
- Dark Mode
- Theme preference saved using Local Storage

---

## 🛠 Built With

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/your-username/proddash.git
```

Go to the project folder

```bash
cd proddash
```

Open the project

```bash
index.html
```

or run it using **Live Server** in VS Code.

## 📦 APIs Used

### Open-Meteo

Provides live weather information.

https://open-meteo.com/

### OpenStreetMap Nominatim

Used for reverse geocoding to display the user's current city.

https://nominatim.openstreetmap.org/

---

## 💾 Data Storage

All user data is stored locally in the browser using **Local Storage**.

This includes:

- Theme preference
- Todo list
- Daily planner
- Daily goals

No external database is required.
